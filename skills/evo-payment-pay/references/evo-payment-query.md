# payment +query

Query payment status by merchant transaction ID.

## Command

```bash
evo-cli payment +query --merchant-tx-id <id>
```

## Required Flags

| Flag | Description |
|------|-------------|
| `--merchant-tx-id` | Merchant transaction ID to query |

## Example

```bash
evo-cli payment +query --merchant-tx-id TX20260401001
```
